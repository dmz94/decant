<h2>Notes</h2>
<ul>
<li>Figures for China exclude <a href="/wiki/Taiwan">Taiwan</a>, and the<a href="/wiki/Special_administrative_regions_of_China">special administrative regions</a>of<a href="/wiki/Hong_Kong">Hong Kong</a>and<a href="/wiki/Macau">Macau</a>.</li>
<li>Based on the best available GDP figure for each country at the time of creation (16 June 2024). Best available GDP figure was defined as the latest available figure from the <a href="/wiki/International_Monetary_Fund">International Monetary Fund</a>(including forecasts for the current year), otherwise the latest available figure from the<a href="/wiki/World_Bank">World Bank</a>, otherwise the latest available figure from the<a href="/wiki/United_Nations_Statistics_Division">United Nations Statistics Division</a>; however, countries that didn&#x27;t have any figure for a year more recent than 2019 were not included at all. For almost all countries, the latest IMF forecast figure for 2024 was used.</li>
<li>The name used in the IMF report is &quot;Taiwan Province of China&quot;.</li>
<li>The name used in the IMF report is &quot;Hong Kong SAR&quot;.</li>
<li>Figures exclude the <a href="/wiki/Autonomous_Republic_of_Crimea">Autonomous Republic of Crimea</a>and<a href="/wiki/Sevastopol">Sevastopol</a>.</li>
<li>The UN figure excludes the autonomous region of <a href="/wiki/Zanzibar">Zanzibar</a>.</li>
<li>The name used in the IMF report is &quot;Macao SAR&quot;.</li>
<li>Data are for the area controlled by the Government of the Republic of Cyprus.</li>
<li>Excludes <a href="/wiki/Abkhazia">Abkhazia</a>and<a href="/wiki/South_Ossetia">South Ossetia</a></li>
<li>Excludes data for <a href="/wiki/Transnistria">Transnistria</a></li>
<li>The name used in the IMF and World Bank reports is &quot;West Bank and Gaza&quot;.</li>
<li>The name used by the UN is &quot;State of Palestine&quot;.</li>
</ul>
